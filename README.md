# StackVault Protocol �

**StackVault** is a decentralized liquidity protocol on Stacks (Bitcoin L2). It allows users to deposit STX into a vault, providing liquidity and earning simulated yield on their assets.

## 🚀 Vision
StackVault demonstrates the power of Bitcoin-backed finance. By leveraging the security of Bitcoin and the programmability of Stacks, we provide a secure environment for liquidity management.

## ✨ Key Features
- **Liquidity Deposits**: Easily deposit STX into the core protocol.
- **Yield Monitoring**: Real-time tracking of vault balances and simulated returns.
- **DeFi Ready**: Built with robust Clarity smart contracts.
- **Premium UX**: A professional-grade dashboard for asset management.

## 🛠️ Tech Stack
- **Frontend**: React, TypeScript, Vite
- **Blockchain**: Stacks (Bitcoin L2)
- **Library**: `@stacks/connect`, `@stacks/transactions`
- **Smart Contracts**: Clarity (DeFi Optimized)
- **Tooling**: Clarinet for smart contract development and testing

## 🏃 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16+)
- [Clarinet](https://github.com/hirosystems/clarinet)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Testing
To run the protocol tests:
```bash
clarinet test
```
